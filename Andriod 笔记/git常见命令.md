###常见6个命令
* git clone：从远程仓库克隆到本地仓库
* checkout：本地仓库检出到工作空间
* pull：远程仓库拉取到工作空间
* add：工作空间添加到暂存区
* commit：暂存区提交到本地仓库
* push：本地仓库推送到远程仓库

###熟练使用的命令
* init：新建一个代码库
* config --list：显示当前的git配置
* rm：删除文件
* branch：列出所有本地分支
* branch -r：列出所有远程分支
* branch -a：列出所有本地分支和远程分支
* tag：列出所有标签
* status：显示所有变更的文件
* log：显示当前分支的版本历史
* remote -v：显示所有远程仓库
* reset [file]：重置暂存区的指定文件，与上次commit保持一致，但工作空间不变
